# 💡 LED Color / Seed

**Switch the SEED** and **LED Color** by _single-touching_ the sensor (TTP-223).

**9 different color/SEED pairs available:**

1. 🐢 Turtle Green
2. ☀️ Sunny Yellow
3. 🍒 Cherry Red
4. 💙 Electric Blue
5. ❄️ Snow White
6. 🔮 Mystic Violet
7. 🔥 Fire Blaze
8. 🌊 Aqua Breeze
9. 🎀 Bubblegum Pink

## LED State Meaning

* `ON`
  * Idle
* `PULSING`
  * Password/OTP ready to type
* `BLINKING` (fast blink)
  * Typing/Encrypting
* `OFF`
  * No power input
